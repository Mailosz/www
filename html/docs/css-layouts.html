CSS Display

Flow layout

This is the original and default layout that an element can have. It can be enabled explicitly by setting display property to "flow".

Flow layout distinguishes two types of elements: inline and block elements.

Inline elements are displayed "in line", along the text. On the other hand, block elements take whole line for themselves. Consider the following example:

<p>This is a block element. It fills the whole line.</p> <p>This is another block element. <mark>It contains an inline element. </mark><b>And another one. </b><i>And one more.</i></p> <span>This is an inline element.</span> <span>This is an inline element too.</span> <p>And a block element.</p>

<p> is an example of a block element, while <mark>, <b>, <i> and <span> are all inline elements.

You can see that each block element takes up a whole line, regardles of how much content it has, and inline elements are displayed next to each other.

Flexbox

Flexbox is a layout that treats all items the same (there aren't a difference between inline and block elements).

It stack elements in a line.

You enable it by setting display: flex;

See flexbox playground to test several CSS properties relevant to flexbox.

Grid

It aligns elements on a grid.

See grid playground to test several CSS properties relevant to flexbox.
